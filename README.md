# Machine-Learning-Exercises

This repository contains nine machine learning exercises completed using Python and Scikit-Learn, covering both supervised and unsupervised learning techniques.

## Projects Included

1.  *Gold Price Prediction*: Linear Regression model to forecast prices based on historical data.
2.  *USA Housing Prediction*: Multiple Linear Regression to predict house prices using feature scaling.
3.  *Ad Clicks Prediction*: Logistic Regression for binary classification of user behavior.
4.  *SVM Classifier*: Support Vector Machine model tuned using GridSearchCV.
5.  *Decision Tree Classifier*: Classification tree visualized for interpretability.
6.  *Random Forest Classifier*: Ensemble method using multiple trees for improved accuracy.
7.  *Naive Bayes Classifier*: Probabilistic model for classification.
8.  *KNN Classifier*: K-Nearest Neighbors model tuned using the elbow method.
9.  *K-Means Clustering*: Unsupervised learning for customer segmentation using the elbow method.

## Technologies Used
* Python
* Scikit-Learn
* Pandas & NumPy
* Matplotlib & Seaborn
